# BUS32120.Group.9.Final.Project
## BUS32120 Spring 2026 Group 9 Final Project
## NHTSA Vehicle Safety Analysis
Can safety statistics be translated into applicable real-world conclusions to assist aspiring car owners in choosing a vehicle to purchase? In this analysis, we seek to link safety, recall, and complaint data from NHTSA to actual investigations and incidents to uncover connections and predictive ability.

This repository contains a .ipynb notebook file with the full analysis, a .html file containing the analysis with outputs, .sql file displaying all SQL queries used for analysis, and a .pdf file with a presentable slide deck containing all plots generated during analysis with brief descriptions.
